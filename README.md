# Assembler

## Universidade de Brasília (UnB) - Software Básico
===

<br/>
To compile:

```sh
g++ montador.cpp -o montador
```

To run:
```
./montador
```

<br/>

## After:
---

To pre-process only, generating a `.pre` file:
```
./montador -p myprogram.asm
```

To pre-process only, generating a `.obj`:
```
./montador -o
```
